Installation

  1. Get Prism Launcher at Prism Launcher Download
  2. Install it following the website instructions
  3. In the launcher, click the 'Add Instance' button
  4. Choose import and paste this link: https://github.com/Shagulka/Grinnell-SMP-Website/raw/main/GrinnellSMP.zip
  5. You can now start the instance and play on the server!
  6. If Prism Launcher asks you for an account, you can add either Microsoft (if you have the license) or offline (you will still be able to play online)

    Bypass Prism Account Requirement

    ⚠️ This is for entertainment purposes only ⚠️

    If you want to bypass the requirement for a real Microsoft account, you can use the following steps:

    1. Install Prism Launcher from here and go through the quick setup.
    2. Close the launcher.
    3. Execute the following command in the terminal appropriate for your OS:
      • Windows CMD: echo {"accounts": [{"entitlement": {"canPlayMinecraft": true,"ownsMinecraft": true},"type": "Offline"}],"formatVersion": 3} > %appdata%/PrismLauncher/accounts.json
      • Linux Shell: echo '{"accounts": [{"entitlement": {"canPlayMinecraft": true,"ownsMinecraft": true},"type": "Offline"}],"formatVersion": 3}' > ~/.local/share/PrismLauncher/accounts.json
      • Linux Shell (flatpak): echo '{"accounts": [{"entitlement": {"canPlayMinecraft": true,"ownsMinecraft": true},"type": "Offline"}],"formatVersion": 3}' > ~/.var/app/org.prismlauncher.PrismLauncher/data/PrismLauncher/accounts.json
      • macOS: echo '{"accounts": [{"entitlement": {"canPlayMinecraft": true,"ownsMinecraft": true},"type": "Offline"}],"formatVersion": 3}' > ~/ Library/Application\ Support/PrismLauncher/accounts.json
    4. Reopen Prism Launcher and create an offline account.
    5. Delete the "No Profile" account.
    6. Set the new offline account as the default.
    7. Enjoy!

Manual Installation

If you prefer to install the mods manually without using Prism Launcher, follow these steps:

  1. Download and install Minecraft from the official Minecraft website.
    Or use any other launcher if you are a sea sailer, for example Legacy Launcher
  2. The version of Minecraft should be 1.20.6
  3. Download and install the latest version of Fabric Loader.
  4. Download the following mods and place them in your .minecraft/mods folder:
    • Simple Voice Chat (required)
    • Just Enough Items (JEI) (recommended)
    • Sodium (recommended)
  5. Launch Minecraft using the Fabric Loader profile.
  6. Add the server using the IP: grinnell.fun.
  7. Join the server and have fun!
